# Digital Payments Analytics

A data analytics project that analyzes digital payment transactions to uncover spending patterns, transaction trends, customer behavior, and payment method insights. The project leverages Python, SQL, Power BI, and data visualization techniques to transform raw transaction data into actionable business intelligence.

## Project Objectives

* Analyze digital payment transaction data
* Identify spending and transaction trends
* Monitor payment method usage
* Generate business insights through dashboards
* Support data-driven decision making

## Technologies Used

* Python
* Pandas
* NumPy
* SQL
* Power BI
* Matplotlib
* Seaborn
